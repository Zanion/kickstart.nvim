## UR-003: View Blocked Issues

**Type:** User Requirement
**Status:** Finalized

### Description
Users can view issues that are currently blocked by other issues using `:BdBlocked` command, with optional filtering by assignee.

### Rationale
Users need visibility into blockers to understand and unblock their workflow.

### Source
Spec Section 1 (Telescope Pickers - Blocked Issues)

### Acceptance Criteria
- [ ] `:BdBlocked` command opens Telescope picker
- [ ] Only displays blocked issues (from `bd blocked`)
- [ ] Same picker interface as `:BdList`
- [ ] `:BdBlocked` supports optional `--assigned <user>` filter
- [ ] Blocked issues display information about what they are blocked by
