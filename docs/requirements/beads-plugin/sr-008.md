## SR-008: Workspace Database Discovery

**Type:** System Requirement
**Status:** Draft

### Description
The plugin must auto-discover the workspace-relative bd database path based on project directory and `.beads/*.db` presence.

### Priority
- [ ] High

### Dependencies
- SR-001

### Constraints
- Must search from current working directory upward for `.beads/` directory
- Must set BD_PATH environment variable or config for bd commands
- Must gracefully handle no database found (use default bd behavior)

### Acceptance Criteria
- [ ] `beads.workspace.find_db()` searches upward from cwd for `.beads/` directory
- [ ] Checks for `.beads/*.db` files to confirm valid beads directory
- [ ] Returns path to use as `--db` flag or BD_PATH env var
- [ ] Caches discovered path per workspace to avoid repeated filesystem checks
- [ ] Configuration option to override auto-discovery with explicit path
