## UR-001: View Issues List with Telescope Picker

**Type:** User Requirement
**Status:** Finalized

### Description
Users can view all open issues using the `:BdList` command, which opens a Telescope picker displaying a fuzzy-searchable list of issues from `bd list`. The picker supports filtering by assignee using `--assigned` flag.

### Rationale
This is the primary entry point for browsing issues. Users need to see all open issues in an interactive, searchable interface.

### Source
Spec Section 1 (Telescope Pickers - List Issues)

### Acceptance Criteria
- [ ] `:BdList` command opens a Telescope picker
- [ ] Picker displays all issues from `bd list --json` output
- [ ] Issues display searchable title, ID, type, state, and labels
- [ ] `:BdList` supports optional `--assigned <user>` filter
- [ ] Selection opens issue in browser via default keymap `<CR>`
- [ ] Keymap `<leader>i` triggers `:BdShow` for selected issue
