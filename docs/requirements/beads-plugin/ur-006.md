## UR-006: Preview Issue Details

**Type:** User Requirement
**Status:** Finalized

### Description
Users can view full issue details in a Neovim vertical split buffer using `:BdShow <issue-id>`.

### Rationale
After selecting an issue from a picker, users need to see complete details before deciding to work on it.

### Source
Spec Section 2 (Issue Preview)

### Acceptance Criteria
- [ ] `:BdShow <id>` opens vertical split (`:vsplit`) with issue details
- [ ] Displays: title, type, state, labels, description, dependencies
- [ ] Buffer is read-only
- [ ] Available via keymap `<leader>i` from picker selection
- [ ] Run `bd show <id> --json` for data source
- [ ] Follow-up: horizontal split `:BdShowH` also available
