## SR-006: Browser Open Action

**Type:** System Requirement
**Status:** Draft

### Description
The plugin must open issue URLs in the system default browser.

### Priority
- [ ] High

### Dependencies
- SR-001

### Constraints
- Must cross-platform: Linux (xdg-open), macOS (open), Windows (start)
- Must retrieve URL via `bd show <id> --url`

### Acceptance Criteria
- [ ] `beads.actions.open_browser(id)` retrieves URL and opens in browser
- [ ] Platform detection uses `vim.fn.has("unix")`, `vim.fn.has("macunix")`, `vim.fn.has("win32")`
- [ ] Error handling: show notification if URL retrieval fails
- [ ] Works from Telescope picker selection via keymap
