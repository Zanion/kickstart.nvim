## SR-004: Query Language Picker

**Type:** System Requirement
**Status:** Draft

### Description
The plugin must implement a query expression picker for advanced bd query language searches.

### Priority
- [ ] High

### Dependencies
- SR-001, SR-002

### Constraints
- Must accept full bd query expression syntax
- Must show user-friendly error for invalid query expressions

### Acceptance Criteria
- [ ] `:BdQuery` opens Telescope picker with input prompt for query expression
- [ ] Executes `bd query <expr> --json` on input confirmation
- [ ] Invalid query expressions show user-friendly error message
- [ ] Results displayed in standard picker format
