## UR-008: View Dependency Graph

**Type:** User Requirement
**Status:** Finalized

### Description
Users can visualize issue dependencies as a graph using `:BdGraph [issue-id]` displayed in a terminal-based float window.

### Rationale
Understanding dependency relationships is crucial for planning work and identifying blockers.

### Source
Spec Section 4 (Dependency Graph)

### Acceptance Criteria
- [ ] `:BdGraph` opens float window with terminal displaying DAG visualization
- [ ] `:BdGraph <id>` shows graph centered on specific issue
- [ ] `:BdGraph --all` shows all open issues grouped by connected component
- [ ] Supports format options: default (DAG), `--box` (ASCII boxes), `--compact` (tree)
- [ ] Float window is closable via `<Esc>` or `q`
- [ ] Window respects stdlib/nvim window dimensions configuration
