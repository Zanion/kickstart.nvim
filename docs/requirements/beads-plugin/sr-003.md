## SR-003: Live Grep Search with Debounce

**Type:** System Requirement
**Status:** Draft

### Description
The plugin must implement Telescope live_grep with debounced bd search for full-text search functionality.

### Priority
- [ ] High

### Dependencies
- SR-001, SR-002

### Constraints
- Must debounce input to avoid excessive bd calls (300ms default)
- Must handle empty search gracefully
- Must provide fallback (Enter key) for immediate search

### Acceptance Criteria
- [ ] `:BdSearch` opens Telescope live_grep picker
- [ ] Input uses 300ms debounce before executing `bd search`
- [ ] Pressing Enter executes immediate search (bypasses debounce)
- [ ] Search results displayed in standard picker format
- [ ] Error handling: message shown if search command fails
