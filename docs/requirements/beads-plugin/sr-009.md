## SR-009: Dependency Graph Float Window

**Type:** System Requirement
**Status:** Draft

### Description
The plugin must display the dependency graph in a terminal-based floating window with proper formatting options.

### Priority
- [ ] High

### Dependencies
- SR-001

### Constraints
- Must use nvim floating terminal (`nvim_open_term` or `vim.term_open`)
- Must support format options: default (DAG), --box, --compact
- Must handle --all flag for showing all connected components

### Acceptance Criteria
- [ ] `beads.graph.show(id, opts)` opens float window with graph
- [ ] float window centered, 80% width, 70% height
- [ ] Executes `bd graph [id] --json` internally to get graph data
- [ ] Uses terminal to render formatted output based on format option
- [ ] <Esc> and `q` keybindings to close float window
- [ ] Window title shows issue ID if provided, or "All Issues" for --all
- [ ] Follows stdlib window layout configuration
