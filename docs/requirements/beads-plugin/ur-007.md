## UR-007: Open Issue in Browser

**Type:** User Requirement
**Status:** Finalized

### Description
Users can open an issue in the default browser using `:BdOpen <issue-id>`.

### Rationale
Some users prefer to interact with issues in the web UI for full editing capabilities.

### Source
Spec Section 3 (Issue Navigation)

### Acceptance Criteria
- [ ] `:BdOpen <id>` opens issue URL in browser via `bd show <id> --url` and system browser handler
- [ ] Available via keymap `<CR>` from picker selection
- [ ] Neovim executes `xdg-open` (Linux), `open` (macOS), or `start` (Windows)
