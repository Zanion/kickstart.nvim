## UR-010: View Database Status

**Type:** User Requirement
**Status:** Finalized

### Description
Users can view database statistics using `:BdStatus` command displayed in a vertical split, showing bd status JSON output and owner info.

### Rationale
Users need visibility into their issue database health and statistics.

### Source
Spec Section 6 (Commands - :BdStatus)

### Acceptance Criteria
- [ ] `:BdStatus` opens vertical split with issue database statistics
- [ ] Displays `bd status --json` output (total, open, closed counts)
- [ ] Displays owner information from `.beads/` directory
- [ ] Buffer is read-only
- [ ] Includes issue counts per state

- SR-001, SR-006, SR-007
