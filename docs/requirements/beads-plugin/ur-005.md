## UR-005: Query Issues with Advanced Query Language

**Type:** User Requirement
**Status:** Finalized

### Description
Users can execute advanced query expressions using the bd query language via `:BdQuery` command with an input prompt.

### Rationale
Power users need complex filtering beyond basic search (e.g., find issues labeled "bug" in "v2.0" milestone).

### Source
Spec Section 1 (Telescope Pickers - Query Issues)

### Acceptance Criteria
- [ ] `:BdQuery` opens Telescope picker with input prompt
- [ ] User can type query expression in prompt
- [ ] Query executes against bd query language
- [ ] Results display matching issues
- [ ] Selection and preview actions work same as `:BdList`
