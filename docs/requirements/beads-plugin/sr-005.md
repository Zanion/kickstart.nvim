## SR-005: Issue Detail View

**Type:** System Requirement
**Status:** Draft

### Description
The plugin must display full issue details in a vertical split buffer showing parsed JSON data from bd show.

### Priority
- [ ] Critical

### Dependencies
- SR-001

### Constraints
- Must format JSON data into human-readable display
- Must use vertical split (`:vsplit`)
- Must support optional horizontal split variant (`:split`)

### Acceptance Criteria
- [ ] `beads.show.issue(id)` opens `:vsplit` with issue details
- [ ] Displays parsed fields: title, type, state, labels, description, dependencies, created/updated dates
- [ ] Dependencies section shows blocking/blocked-by issue references
- [ ] Buffer set to read-only (modifiable=false)
- [ ] Buffer name format: `beads://issue/<id>`
- [ ] Auto-closes on bufleave if configured
- [ ] Help text in buffer footer with keybindings
