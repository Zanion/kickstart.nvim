## UR-009: Plugin Configuration

**Type:** User Requirement
**Status:** Finalized

### Description
Users can configure plugin behavior via `require("beads").setup({})` with workspace-relative bd path discovery.

### Rationale
Users need to customize binary path, default options, and keybindings. Workspace discovery allows different issue DBs per project.

### Source
Spec Section (Configuration)

### Acceptance Criteria
- [ ] Configurable `binary` path to bd executable (default: "bd")
- [ ] Auto-discover `.beads/*.db` path per project (cwd-based)
- [ ] Configurable default options for each command (`list_opts`, `ready_opts`, etc.)
- [ ] Configurable keymaps for picker actions: open (`<CR>`), show (`<leader>i`), close (`q`)
- [ ] All commands respect workspace-relative bd path
