## SR-002: Telescope Picker Integration

**Type:** System Requirement
**Status:** Draft

### Description
The plugin must integrate with Telescope to provide interactive issue pickers with proper display and actions.

### Priority
- [ ] Critical

### Dependencies
- SR-001 (bd CLI invocation)

### Constraints
- Must use Telescope's native picker API
- Must handle large issue lists efficiently (virtual scrolling not required for Phase 1)
- Must support custom attach_mappings for keybindings

### Acceptance Criteria
- [ ] `beads.pickers.list()` creates Telescope picker for `bd list`
- [ ] `beads.pickers.ready()` creates Telescope picker for `bd ready`
- [ ] `beads.pickers.blocked()` creates Telescope picker for `bd blocked`
- [ ] All pickers display: ID, title, type, state, labels (as comma-separated string)
- [ ] Default sorting by title
- [ ] Default prompt prefix shows picker type (e.g., "[Beads] List> ")
- [ ] Results cached briefly (500ms) to avoid redundant bd calls on Telescope refresh
