## UR-004: Search Issues Full-Text

**Type:** User Requirement
**Status:** Finalized

### Description
Users can perform full-text search across all issues using `:BdSearch` command with Telescope live_grep. Uses debounced live search with fallback to prompt for complex queries.

### Rationale
Users need to find issues containing specific text in titles, descriptions, or comments quickly.

### Source
Spec Section 1 (Telescope Pickers - Search Issues)

### Acceptance Criteria
- [ ] `:BdSearch` opens Telescope live_grep picker
- [ ] Search input uses debounce (300ms default) to avoid excessive bd calls
- [ ] Results include matches from issue content (title, description)
- [ ] Fallback: if debounce causes issues, user can press Enter for immediate search
- [ ] Selection and preview actions work same as `:BdList`
