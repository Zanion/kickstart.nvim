# ADR-005: Terminal Preview Strategy

## Status
Proposed

## Context
Telescope pickers support previewers, but terminal buffers require special handling since they aren't traditional file buffers.

## Decision
Implement a custom telescope previewer that:
1. On selection change, look up the agent's terminal in snacks.terminal.list()
2. Get the buffer number for that terminal
3. Read buffer content via `vim.api.nvim_buf_get_lines()`
4. Render in preview window with virtual text styling
5. Refresh on a timer (every 1-2 seconds) while preview is open

## Alternatives Considered

### Option A: Live terminal buffer (CHOSEN)
- Render terminal buffer directly
- Pros:
  - Real-time updates
  - Accurate terminal representation
  - Leverages snacks.nvim API
- Cons:
  - Preview may lag behind real-time
  - Large scrollback could impact performance

### Option B: Periodic snapshot
- Poll terminal to temp file periodically
- Pros:
  - Isolated from terminal state
  - Consistent preview
- Cons:
  - Adds complexity
  - May miss rapid output
  - Temp file management

### Option C: Scrollback via RPC
- Request scrollback from terminal via RPC
- Pros:
  - Clean separation
  - On-demand fetching
- Cons:
  - Requires agent support
  - More complex implementation

## Implementation

### Previewer Structure

```lua
local TerminalPreviewer = {}

function TerminalPreviewer:new(opts)
  local self = setmetatable({}, { __index = self })
  self.max_lines = opts.max_lines or 1000
  self.refresh_interval = opts.refresh_interval or 1000
  self.timer = nil
  return self
end

function TerminalPreviewer:get_terminal_bufnr(session_id)
  local snacks = require("snacks")
  local terms = snacks.terminal.list()
  
  for _, term in pairs(terms) do
    if term.id == session_id then
      return term.buf
    end
  end
  
  return nil
end

function TerminalPreviewer:render(preview_bufnr, preview_winid, entry)
  local bufnr = self:get_terminal_bufnr(entry.value.session_id)
  
  if not bufnr or not vim.api.nvim_buf_is_valid(bufnr) then
    vim.api.nvim_buf_set_lines(
      preview_bufnr, 0, -1, false,
      { "Terminal not running", "", "Press <CR> to restart" }
    )
    return
  end
  
  -- Get terminal content
  local lines = vim.api.nvim_buf_get_lines(bufnr, 0, -1, false)
  
  -- Take last N lines
  local start = math.max(0, #lines - self.max_lines)
  local preview_lines = vim.list_slice(lines, start + 1, #lines)
  
  -- Format and strip ANSI
  local formatted = self:strip_ansi(preview_lines)
  
  -- Add header
  local header = {
    string.format("=== %s | %s | %s ===",
      entry.value.agent,
      entry.value.bead_id,
      entry.value.status
    ),
    "",
  }
  
  vim.api.nvim_buf_set_lines(
    preview_bufnr, 0, -1, false,
    vim.list_extend(header, formatted)
  )
  
  -- Scroll to bottom
  vim.api.nvim_win_set_cursor(
    preview_winid,
    { #formatted + #header, 0 }
  )
end

function TerminalPreviewer:strip_ansi(lines)
  local result = {}
  for _, line in ipairs(lines) do
    -- Remove ANSI escape codes
    local clean = line:gsub("\27%[%d*m", "")
    table.insert(result, clean)
  end
  return result
end
```

### Telescope Integration

```lua
require("telescope").register_previewer({
  name = "work_dispatch_terminal",
  
  setup = function(state)
    -- Initialize previewer
    state.previewer = TerminalPreviewer:new({
      max_lines = 500,
      refresh_interval = 1000,
    })
  end,
  
  teardown = function(state)
    -- Clean up timer
    if state.previewer.timer then
      state.previewer.timer:stop()
      state.previewer.timer:close()
    end
  end,
  
  preview_fn = function(self, entry, status)
    -- Initial render
    self.previewer:render(
      status.preview_bufnr,
      status.preview_winid,
      entry
    )
    
    -- Start live refresh
    self.previewer:start_refresh(
      status.preview_bufid,
      entry.value.session_id
    )
  end,
})
```

## Live Refresh

```lua
function TerminalPreviewer:start_refresh(bufnr, session_id)
  if self.timer then
    self.timer:stop()
    self.timer:close()
  end
  
  self.timer = vim.loop.new_timer()
  
  local preview_bufnr = bufnr
  local preview_winid = self:get_preview_winid()
  
  self.timer:start(
    0,
    self.refresh_interval,
    vim.schedule_wrap(function()
      if not vim.api.nvim_win_is_valid(preview_winid) then
        self:stop_refresh()
        return
      end
      
      self:render(preview_bufnr, preview_winid, {
        value = { session_id = session_id }
      })
    end)
  )
end
```

## Performance Considerations

| Aspect | Limit | Rationale |
|--------|-------|-----------|
| Max lines shown | 1000 | Memory, readability |
| Refresh interval | 1000ms | Balance between responsiveness and performance |
| Buffer validation | Each refresh | Prevent errors on closed buffers |

## Consequences

### Easier
- Leverages existing snacks.nvim API
- No external dependencies
- Relatively simple implementation

### Harder
- Preview may lag behind real-time
- Large scrollback could impact performance
- Buffer may be invalid if terminal closed
