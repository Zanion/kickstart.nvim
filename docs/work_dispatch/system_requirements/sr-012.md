# SR-012: Agent Focus Action

## Status
Draft

## Priority
High

## Dependencies
- SR-007: Agent Session Management
- SR-010: Agent Picker UI

## Description
Action to focus/open a selected agent's terminal.

## Constraints

### Behavior
- `<CR>` key focuses agent terminal
- Toggle if already focused
- Spawn if not running

### Terminal Management
- Use snacks.nvim terminal toggle
- Maintain session reference

## Focus Flow

```
User presses <CR> on picker entry
         ↓
Check if session exists
         ↓
    ┌────┴────┐
    │         │
  Yes         No
    │         │
    ↓         ↓
Terminal     Spawn agent
running?     in worktree
    │
┌───┴───┐
│       │
Yes     No
│       │
↓       ↓
Toggle  Create new
focus   session
```

## Functions

### `actions.focus(entry)`
Main focus action.

**Parameters:**
- `entry` (table, required) - Picker entry

**Returns:** boolean success

### `actions.focus_or_spawn(worktree_id)`
Focuses existing terminal or spawns new one.

**Parameters:**
- `worktree_id` (string, required)

**Returns:**
```lua
{
  success = true,
  action = "focus" | "spawn",
  session_id = "uuid",
}
```

## Implementation

```lua
local actions = {}

function actions.focus(entry)
  local worktree_id = entry.value.id
  
  -- Check if session exists and is running
  local session = session.get_by_worktree(worktree_id)
  
  if session and terminal.is_running(session.terminal_id) then
    -- Toggle focus
    return actions.toggle_focus(session)
  else
    -- Spawn new agent
    return actions.spawn_agent(entry.value)
  end
end

function actions.toggle_focus(session)
  local snacks = require("snacks")
  local term = snacks.terminal.list()[session.terminal_id]
  
  if term then
    term:toggle()
    return { success = true, action = "focus" }
  else
    -- Terminal was closed, respawn
    return actions.spawn_agent(session)
  end
end

function actions.spawn_agent(worktree)
  -- Get agent command
  local cmd = agent.get_command(worktree.agent)
  
  -- Build options
  local opts = {
    cwd = worktree.path,
    env = {
      NVIM_LISTEN_ADDRESS = worktree.nvim_socket,
      BEAD_ID = worktree.bead_id,
    },
    win = { style = "float", border = "rounded" },
  }
  
  -- Spawn terminal
  local term = snacks.terminal.toggle(cmd, opts)
  
  -- Update registry
  registry.update(worktree.id, {
    terminal_id = term.id,
    terminal_buf = term.buf,
    status = "running",
  })
  
  return { success = true, action = "spawn", session_id = worktree.id }
end
```

## Session Recovery

If terminal closed unexpectedly:

```lua
function session.recover(worktree_id)
  local worktree = registry.get(worktree_id)
  if not worktree then return nil end
  
  -- Check if terminal is still running
  local snacks = require("snacks")
  local terms = snacks.terminal.list()
  
  for _, term in pairs(terms) do
    if term.buf == worktree.terminal_buf then
      return term  -- Terminal still alive
    end
  end
  
  -- Terminal died, update status
  registry.update(worktree_id, {
    status = "paused",
    terminal_buf = nil,
    terminal_id = nil,
  })
  
  return nil
end
```

## Acceptance Criteria

- [ ] `<CR>` key focuses agent terminal
- [ ] If terminal exists: bring to front or toggle
- [ ] If terminal doesn't exist: spawn in worktree
- [ ] Session state updated appropriately

## Edge Cases

| Case | Behavior |
|------|----------|
| Terminal frozen | Allow force kill option |
| Terminal crashed | Mark as paused, offer respawn |
| Multiple terminals same worktree | Focus most recent |
| Terminal in different Neovim instance | Show warning, offer to close |
