# SR-005: Dispatch Command

## Status
Draft

## Priority
Critical

## Dependencies
- SR-001: Worktree Directory Management
- SR-002: Worktree Creation and Lifecycle
- SR-003: Worktree Registry
- SR-004: Beads Integration Layer

## Description
Command to dispatch a bead to a specific agent.

## User Flow
1. User invokes picker: `bd dispatch` or keybind
2. Picker shows ready beads from `bd ready`
3. User selects bead
4. Picker shows available agents
5. User selects agent
6. System creates worktree, spawns agent

## Functions

### `dispatch.pick_bead()`
- Opens telescope picker for ready beads
- Returns: selected bead object or nil

### `dispatch.pick_agent()`
- Opens telescope picker for available agents
- Returns: selected agent name or nil

### `dispatch.execute(bead_id, agent_name)`
Main dispatch function.

**Parameters:**
- `bead_id` (string, required) - Bead to dispatch
- `agent_name` (string, required) - Agent type (gemini/cursor/claude/opencode)

**Returns:**
```lua
{
  success = true,
  worktree = { -- worktree info },
  bead = { -- bead details },
}
```

**Error Returns:**
```lua
{
  success = false,
  error = "error message",
  code = "ERROR_CODE",
}
```

### `dispatch.cancel(worktree_id)`
- Cancels a dispatch in progress
- Cleans up worktree if created
- Returns: boolean success

## Dispatch Process

```lua
function dispatch.execute(bead_id, agent_name)
  -- 1. Fetch bead details
  local bead, err = beads.show(bead_id)
  if not bead then return nil, err end

  -- 2. Create worktree
  local worktree, err = worktree.create({
    bead_id = bead_id,
    bead_title = bead.title,
    agent = agent_name,
  })
  if not worktree then return nil, err end

  -- 3. Register worktree
  registry.add(worktree)

  -- 4. Inject bead context
  beads_context.inject(worktree.path, bead)

  -- 5. Spawn agent
  local session_id, err = agent.spawn(worktree, agent_name)
  if not session_id then
    registry.update(worktree.id, { status = "failed" })
    return nil, err
  end

  -- 6. Update registry
  registry.update(worktree.id, {
    session_id = session_id,
    status = "running",
  })

  -- 7. Claim bead
  beads.claim(bead_id)

  return { worktree = worktree, bead = bead }
end
```

## Acceptance Criteria

- [ ] `require("work_dispatch").dispatch(bead_id, agent_name)`
- [ ] Create worktree with bead-scoped branch
- [ ] Spawn agent terminal with bead context
- [ ] Update bead state to "in_progress" with agent metadata
- [ ] Return worktree info to caller

## Error Handling

| Error | Code | Action |
|-------|------|--------|
| Bead not found | `BEAD_NOT_FOUND` | Return error, no cleanup needed |
| Worktree exists | `WORKTREE_EXISTS` | Offer to create parallel |
| Git error | `GIT_ERROR` | Clean up partial worktree |
| Agent spawn failed | `SPAWN_FAILED` | Remove worktree, unclaim bead |
| IPC setup failed | `IPC_ERROR` | Continue without IPC |

## Configuration

```lua
require("work_dispatch").setup({
  dispatch = {
    auto_claim = true,      -- Auto-claim bead on dispatch
    auto_focus = true,      -- Focus terminal after dispatch
    confirm_parallel = true, -- Confirm before parallel dispatch
  }
})
```
