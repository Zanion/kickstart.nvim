# SR-006: Bead Context Injection

## Status
Draft

## Priority
High

## Dependencies
- SR-004: Beads Integration Layer
- SR-005: Dispatch Command

## Description
Inject bead context and instructions into the agent's worktree workspace.

## Constraints

### File Location
- `BEADS_CONTEXT.md` in worktree root
- Overwrites existing if present

### Format
- Markdown for readability
- Structured sections for parsing

## Functions

### `beads_context.inject(worktree_path, bead)`
**Parameters:**
- `worktree_path` (string, required)
- `bead` (table, required) - Full bead object from beads.show()

**Returns:** boolean success

**Side Effects:**
- Creates `BEADS_CONTEXT.md` in worktree
- Copies relevant files from main repo if configured

### `beads_context.generate(bead)`
**Parameters:**
- `bead` (table, required)

**Returns:** string (markdown content)

### `beads_context.get(worktree_path)`
**Parameters:**
- `worktree_path` (string, required)

**Returns:** bead table or nil

## Implementation

```lua
function beads_context.inject(path, bead)
  local context_path = vim.fn.joinpath(path, "BEADS_CONTEXT.md")
  local content = beads_context.generate(bead)
  
  local file, err = io.open(context_path, "w")
  if not file then return false, err end
  file:write(content)
  file:close()
  
  return true
end
```
