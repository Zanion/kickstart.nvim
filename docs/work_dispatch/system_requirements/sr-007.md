# SR-007: Agent Session Management

## Status
Draft

## Priority
Critical

## Dependencies
- Existing `custom/agent.lua`

## Description
Manage agent terminal sessions and track their state using snacks.nvim.

## Constraints

### Terminal Management
- Use snacks.nvim for terminal management
- Track session independently from worktree
- Handle terminal close events

### Session Lifecycle
- Session tied to worktree and bead
- Multiple sessions possible per bead (parallel)
- Session can be resumed after terminal close

## Session Object

```lua
{
  id = "session-uuid",
  worktree_id = "worktree-uuid",
  bead_id = "bd-42",
  agent = "gemini",
  terminal_buf = 123,       -- snacks terminal buffer number
  terminal_id = "term-1",  -- snacks terminal ID
  pid = 12345,              -- process ID
  nvim_socket = "/tmp/nvim-socket-123",
  status = "running",
  started_at = "ISO8601",
}
```

## Functions

### `agent.spawn(worktree, agent_name)`
**Parameters:**
- `worktree` (table, required) - Worktree info from registry
- `agent_name` (string, required) - Agent type

**Returns:**
```lua
{
  session_id = "uuid",
  terminal_id = "term-1",
  nvim_socket = "/path/to/socket",
}
```

**Process:**
1. Create unique nvim socket for IPC
2. Build agent command with socket path
3. Spawn terminal in worktree directory
4. Register session in registry
5. Set up IPC handlers

### `agent.list()`
Returns all active sessions.

### `agent.get(session_id)`
Returns session by ID or nil.

### `agent.get_by_worktree(worktree_id)`
Returns sessions for a worktree.

### `agent.focus(session_id)`
Brings terminal to front or toggles.

### `agent.kill(session_id)`
Kills agent process gracefully.

### `agent.close(session_id)`
Closes terminal and marks session ended.

## Terminal Spawn Command

```lua
local function build_agent_cmd(session)
  local nvim_addr = session.nvim_socket
  local agent_cmd = agent_configs[session.agent].cmd
  
  return string.format(
    "NVIM_LISTEN_ADDRESS=%s %s",
    vim.fn.shellescape(nvim_addr),
    agent_cmd
  )
end
```

## Event Handlers

### On Terminal Close
```lua
snacks.terminal.on_close = function(term)
  local session = find_session_by_terminal(term.id)
  if session then
    registry.update(session.worktree_id, {
      status = "paused",
    })
  end
end
```

## Acceptance Criteria

- [ ] Spawn agent in worktree directory
- [ ] Pass NVIM_LISTEN_ADDRESS for IPC

## Integration with custom/agent.lua

```lua
-- Extend agent module
local agent = require("custom.agent")

-- Add worktree-aware spawning
agent.spawn_worktree = function(worktree, agent_name)
  local opts = {
    cwd = worktree.path,
    env = {
      NVIM_LISTEN_ADDRESS = worktree.nvim_socket,
      BEAD_ID = worktree.bead_id,
      WORKTREE_ID = worktree.id,
    }
  }
  -- Use agent-specific command
  local cmd = get_agent_cmd(agent_name)
  return snacks.terminal.toggle(cmd, opts)
end
```
