## Description

## Constraints

- Floating layout
- Custom previewer

## Functions

### `picker.open()`
Opens the agent picker.

**Parameters:**
- `opts.filter` (table, optional) - Filter criteria
  - `agent` (string) - Filter by agent type
  - `status` (string) - Filter by status
  - `bead_id` (string) - Filter by bead

**Returns:** nothing (opens picker)

### `picker.refresh()`
Refreshes the picker if open.

require("telescope").setup({

## Entry Maker

```lua
local function entry_maker(entry)
  local icon = agent_icons[entry.agent] or "○"
  local status_icon = status_icons[entry.status] or "○"
  local needs_input_mark = entry.needs_input and "⚠" or " "
  
  return {
    value = entry,
    display = string.format(
      "%s %-8s │ %-6s │ %-30s │ %s %s",
      icon,
      entry.agent,
      entry.bead_id,
      entry.name,
      status_icon,
      needs_input_mark
    ),
    ordinal = string.format(
      "%s %s %s %s",
      entry.agent,
      entry.bead_id,
      entry.name,
      entry.status
    ),
  }
end
```

## Actions

### Focus Agent (<CR>)
- Opens/focuses agent terminal
- If terminal closed, respawns

### Merge (<leader>m)
- Triggers merge action
- Shows confirmation if uncommitted changes

### Reject (<leader>r)
- Triggers reject action
- Shows confirmation

### Reset (<leader>R)
- Triggers reset action
- Available for rejected/paused agents

## Sorting

Default sort order:
1. Needs input (by wait time, oldest first)
2. Running (by priority)
3. Paused (by creation time)
4. Done/Rejected (by creation time)

## Filters

| Filter | Key | Description |
|--------|-----|-------------|
| By agent | `<leader>fa` | Show only specific agent |
| By status | `<leader>fs` | Show only status |
| By bead | `<leader>fb` | Show only bead |
| Clear filters | `<leader>fc` | Clear all filters |
