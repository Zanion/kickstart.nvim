# SR-013: Merge Action

## Status
Draft

## Priority
Critical

## Dependencies
- SR-003: Worktree Registry
- SR-004: Beads Integration Layer

## Description
Merge agent work by creating a PR from the worktree branch.

## Constraints

### Pre-Merge Requirements
- Branch must be pushed to remote
- No uncommitted changes (auto-commit)

### PR Creation
- Use `gh` CLI or `git` commands
- Title and body must follow format

## Merge Process

```lua
local function merge(worktree_id)
  local worktree = registry.get(worktree_id)
  if not worktree then return error("Worktree not found") end
  
  -- 1. Change to worktree directory
  local cwd = vim.fn.getcwd()
  vim.fn.chdir(worktree.path)
  
  -- 2. Check for uncommitted changes
  local has_changes = git.has_uncommitted()
  if has_changes then
    local commit_msg = format_commit_message(worktree)
    git.commit(commit_msg)
  end
  
  -- 3. Push branch to remote
  local push_result = git.push(worktree.branch)
  if not push_result.success then
    return error("Push failed: " .. push_result.error)
  end
  
  -- 4. Create PR
  local pr_url = gh.create_pr({
    title = format_pr_title(worktree),
    body = format_pr_body(worktree),
    head = worktree.branch,
    base = "master",  -- or main
  })
  
  -- 5. Close bead
  beads.close(worktree.bead_id, "Implemented")
  
  -- 6. Update registry
  registry.update(worktree_id, {
    status = "done",
    merge_commit = push_result.commit,
    pr_url = pr_url,
  })
  
  -- 7. Restore cwd
  vim.fn.chdir(cwd)
  
  return { success = true, pr_url = pr_url }
end
```

## Git Functions

### `git.has_uncommitted()`
Returns boolean.

### `git.commit(message)`
Commits staged and unstaged changes.

### `git.push(branch)`
Pushes branch to remote.

```lua
local function git.push(branch)
  local result = vim.fn.system({
    "git", "push", "-u", "origin", branch
  })
  
  if vim.v.shell_error ~= 0 then
    return {
      success = false,
      error = result,
    }
  end
  
  -- Extract commit hash
  local commit = result:match("(%x+)")
  
  return { success = true, commit = commit }
end
```

## PR Creation

### `gh.create_pr(opts)`
```lua
local function gh.create_pr(opts)
  local cmd = {
    "gh", "pr", "create",
    "--title", opts.title,
    "--body", opts.body,
    "--head", opts.head,
    "--base", opts.base,
    "--fill",  -- Auto-fill from commits
  }
  
  local result = vim.fn.system(cmd)
  
  if vim.v.shell_error ~= 0 then
    return nil, result
  end
  
  -- Extract PR URL
  local url = result:match("https://github.com/[^\n]+")
  return url
end
```

## Message Formats

### Commit Message
```
feat({bead_id}): {short_description}

{bead_title}

Closes #{bead_number}
```

### PR Title
```
feat({bead_id}): {title}
```

### PR Body
```markdown
## Summary
<!-- Auto-generated from bead -->

## Changes
<!-- From commit messages -->

## Testing
- [ ] Tests added/updated
- [ ] Manual testing completed

## Notes
<!-- Any additional context -->

Closes #{bead_number}
```

## Notification

On merge success:
```lua
require("notify")(
  string.format("PR created: %s", pr_url),
  "info",
  {
    title = "Merge Complete",
    actions = {
      "Open PR",
      "Dismiss",
    },
    callback = function(notif, action)
      if action == "Open PR" then
        vim.fn.system({ "open", pr_url })
      end
    end,
  }
)
```

## Acceptance Criteria

- [ ] Push branch: `git push -u origin {branch}`
- [ ] Create PR: `gh pr create --title "feat: {bead-title}" --body "Closes #{bead-id}"`
- [ ] Update bead to "done" state: `bd close {id} --reason "Implemented"`
- [ ] Show success notification with PR URL
- [ ] Optionally close terminal after merge

## Error Handling

| Error | Code | Recovery |
|-------|------|----------|
| Push rejected | `PUSH_REJECTED` | Pull/rebase, retry |
| PR exists | `PR_EXISTS` | Return existing PR URL |
| Bead already closed | `BEAD_CLOSED` | Confirm before overwriting |
| `gh` not installed | `NO_GH_CLI` | Show setup instructions |
