# SR-011: Terminal Preview Component

## Status
Draft

## Priority
High

## Dependencies
- SR-007: Agent Session Management

## Description
Live preview of agent terminal output in the picker preview pane.

## Constraints

### Preview Method
- Access terminal buffer via snacks.nvim API
- Refresh on selection change and timer
- Handle invalid buffers gracefully

### Performance
- Limit scrollback lines (default: 1000)
- Debounce refresh calls
- Async buffer reads

## Preview Strategy

```lua
local Previewer = {
  new = function(opts)
    return setmetatable({
      timeout = opts.timeout or 30000,
      max_lines = opts.max_lines or 1000,
      refresh_interval = opts.refresh_interval or 1000,
    }, { __index = Previewer })
  end,
}
```

## Functions

### `previewer.preview(entry)`
Called when entry is selected.

**Parameters:**
- `entry` (table, required) - Picker entry

**Process:**
1. Get session by worktree_id
2. Get terminal buffer number
3. Read buffer lines
4. Format for display
5. Render in preview window

### `previewer.render_terminal(bufnr, winid)`
Renders terminal buffer content.

```lua
function Previewer:render_terminal(bufnr, winid)
  if not vim.api.nvim_buf_is_valid(bufnr) then
    return "Terminal buffer no longer valid"
  end
  
  -- Get lines from buffer
  local lines = vim.api.nvim_buf_get_lines(
    bufnr,
    0,
    -1,
    false
  )
  
  -- Take last N lines
  local start = math.max(0, #lines - self.max_lines)
  local preview_lines = vim.list_slice(lines, start + 1, #lines)
  
  -- Format with ANSI color support
  local formatted = self.format_ansi(preview_lines)
  
  -- Set buffer content
  vim.api.nvim_buf_set_lines(bufnr, 0, -1, false, formatted)
  
  -- Enable scrolling
  vim.api.nvim_win_set_cursor(winid, { #formatted, 0 })
end
```

## ANSI Color Handling

Terminal output may contain ANSI escape codes:

```lua
function Previewer:format_ansi(lines)
  local formatted = {}
  for _, line in ipairs(lines) do
    -- Strip ANSI codes for plain text
    local clean = line:gsub("\27[%d+m", "")
    table.insert(formatted, clean)
  end
  return formatted
end
```

## Live Refresh

```lua
local refresh_timer

function Previewer:start_live_refresh(winid, session_id)
  refresh_timer = vim.loop.new_timer()
  refresh_timer:start(
    0,
    self.refresh_interval,
    vim.schedule_wrap(function()
      if vim.api.nvim_win_is_valid(winid) then
        self:refresh_preview(winid, session_id)
      else
        self:stop_live_refresh()
      end
    end)
  )
end

function Previewer:stop_live_refresh()
  if refresh_timer then
    refresh_timer:stop()
    refresh_timer:close()
    refresh_timer = nil
  end
end
```

## Preview States

| State | Display |
|-------|---------|
| Loading | `"Loading preview..."` |
| Valid buffer | Terminal scrollback |
| Terminal closed | `"Terminal closed. Press <CR> to restart."` |
| No session | `"No active session"` |
| Error | `"Error: <message>"` |

## Telescope Integration

```lua
require("work_dispatch.picker").previewer = function()
  return require("telescope.previewers").new_buffer_previewer({
    get_buffer_by_name = function(_, entry)
      return "work_dispatch:" .. entry.value.id
    end,
    
    define_preview = function(self, entry, _)
      local preview_bufnr = self.state.bufnr
      local preview_winid = self.state.winid
      
      -- Start preview
      previewer:preview(entry.value)
      
      -- Start live refresh
      previewer:start_live_refresh(preview_winid, entry.value.id)
      
      -- Clean up on close
      vim.api.nvim_buf_attach(
        preview_bufnr,
        false,
        {
          on_detach = function()
            previewer:stop_live_refresh()
          end,
        }
      )
    end,
  })
end
```

## Acceptance Criteria

- [ ] Implement custom telescope previewer
- [ ] Read terminal buffer content via `snacks.terminal.list()`
- [ ] Display scrollback buffer in preview
- [ ] Handle invalid/closed buffers gracefully
- [ ] Auto-refresh preview on interval (1-2 seconds)

## Performance Considerations

- **Large output**: Truncate to last N lines
- **Frequent updates**: Debounce to max 1/second
- **Memory**: Don't cache full scrollback
- **Buffer invalidation**: Handle gracefully
